# SCORE FINDER
## 1. 프로그램 목적
사설 전투력 점수 산정 사이트 점수를 확인하는 프로그램입니다.

공식 점수가 나오면 해당 점수로 확인하시기 바랍니다.

아툴 : 지원중

아온이 : 암호화로 인해 지원할수 없음

## 2. 설치방법
### 2.1 python 설치
이미지 인식을 위해 python 설치가 필요합니다.

기존에 설치되어 있는분은 생략하셔도 됩니다. 

python 버전은 3.10 이상이 필요합니다. 이전인 분들은 설치해주세요.

windows 기준 : (https://www.python.org/ftp/python/3.14.3/python-3.14.3-amd64.exe)

위의 링크를 클릭하여 파일을 다운로드 받고 실행해주세요.

실행하면 아래와 같은 창이 나올텐데 

<img width="654" height="414" alt="스크린샷 2026-02-08 134726" src="https://github.com/user-attachments/assets/2162a78f-bfa3-4156-8535-b5d11a444840" />

설치 창 맨 밑에 **"Add python.exe to PATH"** 반드시 선택하고 설치해주세요.

설치가 완료되었으면 창을 닫아주세요.

### 2.2 파이썬 라이브러리 설치

사용할 이미지 인식 프로그램을 설치해야 합니다.

먼저 윈도우 키를 누르고 cmd를 검색해서 명령프롬프트 창을 실행시켜 주세요.

<img width="1247" height="601" alt="스크린샷 2026-02-08 135151" src="https://github.com/user-attachments/assets/3d4b75f3-8638-403f-952a-788f5f2e0596" />

위의 화면에 다음과 같이 명령어를 입력해야 합니다.

그래픽카드가 엔비디아 40, 50시리즈라면 13.0 버전을 설치해주시고 10, 20 시리즈라면 12.8 버전을 선택해주세요.

본인의 그래픽카드가 30시리즈라면 둘 중 아무거나 설치해주세요.

12.8 pytorch 명령어 : **pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu128**

13.0 pytorch 명령어 : **pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu130**

알맞게 입력해주시면 "Looking in indexes: https://download.pytorch.org/whl/cu130..." 이런식으로 글이 쭉 올라옵니다.

각자 버전에 맞게 설치해주셨으면 두번째로 명령어를 실행해야 합니다.

명령 프롬프트 창에 **pip install easyocr**을 입력해주세요.

알맞게 입력해주시면 "Collecting easyocr..." 이런식으로 글이 쭉 올라옵니다.

모두 설치하셨으면 창을 닫아주시면 됩니다.

### 2.3 프로그램 설치

[Release](https://github.com/uuuusssseeeerrrr/scoreFinder/releases) 링크에 접속해서 프로그램을 다운받아 설치해주시면 됩니다.

프로그램 설치해주시고 바탕화면에 바로가기 아이콘이 생기면 완료입니다.

프로그램 최초 실행시 이미지 모델 로딩으로 인해 시간이 다소 소요될 수 있습니다.

## 3. 사용방법

### 3.1 원리

성역 파티창 8명이 뜨는곳을 부분 이미지 인식 시켜서 사용자명을 인식합니다.

아이온2 화면 안에서 밑의 화면을 인식합니다. 모니터 내 해당 파티창 세로로 아무 글씨도 없도록 해주세요.

<img width="865" height="585" alt="partyMember" src="https://github.com/user-attachments/assets/de18e08d-9ee7-4170-b8b7-6ced79a16e62" />

파티원을 인식하면 다음과 같이 데이터를 가져옵니다.

이미지 인식상 파티원 이름 및 서버이름이 다를 수 있는데 해당 파티원 이름을 클릭하셔서 알맞게 수정해주세요.

캐릭터명[서버명]의 우측 색상은 다음과 같은 뜻입니다.

- 초록색 : 컴퓨터가 90% 정확도로 읽은 값

- 주황색 : 컴퓨터가 70% 정확도로 읽은 값

- 빨간색 : 컴퓨터가 70% 미만의 정확도로 읽은 값

## 4. 주의 및 안내사항

### 안내사항

- 실행이 안되거나 실행을 했는데 정상적으로 작동이 불가능합니다.
   
프로그램 인식을 제대로 못하거나 실행은 되었는데 아무반응이 없을경우 **C:\Users\사용자\AppData\Local\aion2scf\logs** 폴더 내 파일을 첨부하여 주시거나 파일내 내용을 첨부하여 주시면 오류를 고치는데 도움이 됩니다.

- python이 설치되어 있는것 같은데 확인은 어떻게 하나요?

명령프롬프트 창에 **python --version** 눌러주시면 버전확인이 가능합니다.

- python이 설치되어 있는데 제대로 실행이 안됩니다.

-- path 설정 : **python path 설정**으로 구글에 검색하셔서 보고 따라하시면 됩니다.

-- 버전업 : 삭제하고 다시설치해주세요.

- 이미지 인식이 느립니다.

이미지 인식은 그래픽카드의 처리속도에 따라 달라집니다.

- AMD 그래픽카드는 지원을 안하나요?

CPU나 호환되는 프로그램을 설치하면 될 것 같은데 일단 시간이 없어서 nvidia만 지원하는것으로 업로드 했습니다.

- 그래픽 디자인이 어디서 많이 본 것 같아요.

미터기 디자인 가져와서 재활용 했습니다.

- 캐릭터명이랑 서버를 정상적으로 입력했는데 점수가 정상적으로 불러와지지 않아요.

사이트에서 불러오는 방법이 있는데 해당 불러오는 방법이 변경되었을경우 불러와지지 않습니다. 시간되는데로 빠르게 수정하겠습니다.

- 점수나 전투력이 현재랑 달라요.

업데이트는 직접하셔야 합니다. 해당 사이트에서 검색하셔서 최신버전으로 업데이트 해주세요.

- 서버명은 어떻게 찾나요?
  
상대방 유저의 서버명 두글자를 입력하시면 됩니다.
